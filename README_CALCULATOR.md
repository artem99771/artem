# Программа гидравлического расчета тепловой сети

## Описание
Программа для выполнения гидравлического расчета тепловых сетей. Рассчитывает потери давления на трение и местные сопротивления.

## Функциональность
- Расчет потерь давления в трубопроводах тепловой сети
- Учет линейных и местных потерь
- Расчет в зависимости от температуры теплоносителя
- Определение режима течения (ламинарный, турбулентный)
- Учет дополнительных сопротивлений (повороты, арматура)

## Требования
- Python 3.x
- Tkinter (обычно входит в стандартную поставку Python)

> **Примечание для Linux систем**: Если возникает ошибка `ImportError: libtk8.6.so: cannot open shared object file`, установите пакет tkinter:
> ```bash
> # Ubuntu/Debian:
> sudo apt-get install python3-tk
> 
> # CentOS/RHEL/Fedora:
> sudo dnf install tkinter
> # или
> sudo yum install tkinter
> ```

## Установка и запуск

### Вариант 1: Прямой запуск
```bash
python3 thermal_network_calculator.py
```

### Вариант 2: Использование скрипта запуска
```bash
python3 run_calculator.py
```

### Вариант 3: Установка через установочный скрипт
```bash
python3 install_calculator.py
python3 thermal_network_calculator.py
```

## Использование
1. Введите параметры трубопровода:
   - Расход теплоносителя (м³/ч)
   - Диаметр трубы (мм)
   - Длина участка (м)
   - Шероховатость трубы (мм)
   - Температура теплоносителя (°C)
   - Дополнительные параметры (местные сопротивления, повороты, арматура)

2. Нажмите кнопку "Рассчитать"

3. Результаты отобразятся в нижней части окна

## Результаты расчета
- Плотность и вязкость воды при заданной температуре
- Скорость потока
- Число Рейнольдса и режим течения
- Коэффициент трения
- Потери давления (линейные и местные)
- Общие потери давления в Паскалях, кПа и метрах водяного столба

## Ярлык для запуска
Файл `thermal_network_calculator.desktop` может использоваться как ярлык для запуска приложения в графической среде Linux.
