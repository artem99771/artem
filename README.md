# artem

## Программа гидравлического расчета тепловой сети

Создана настольная программа для гидравлического расчета тепловой сети с графическим интерфейсом.

### Файлы проекта:
- `thermal_network_calculator.py` - основной файл программы с GUI
- `run_calculator.py` - скрипт для запуска приложения
- `install_calculator.py` - скрипт установки и проверки зависимостей
- `thermal_network_calculator.desktop` - ярлык для запуска приложения
- `README_CALCULATOR.md` - подробная документация по использованию
- `calculator_icon.png` - файл иконки (заглушка)

### Особенности программы:
- Графический интерфейс с использованием Tkinter
- Расчет потерь давления в тепловых сетях
- Учет линейных и местных потерь
- Возможность учета температуры теплоносителя
- Расчет числа Рейнольдса и режима течения
- Дружелюбный интерфейс с меню и результатами в текстовом виде
